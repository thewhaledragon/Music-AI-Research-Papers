# Music Information Retrieval (MIR)

A foundational research field focusing on **extracting information from music data**.

### 1 Audio Analysis

* beat tracking
* tempo detection
* pitch detection
* onset detection
* key detection
* chord recognition

---

### 2 Music Classification

* genre classification
* instrument recognition
* mood/emotion recognition
* style classification
* artist identification

Example: deep learning models classify songs into genres using extracted audio features.

---

### 3 Music Structure Analysis

* segmentation
* chorus detection
* structure labeling

---

### 4 Music Similarity & Recommendation

* playlist generation
* music similarity search
* recommendation systems

---

### 5 Lyrics Analysis

* lyric classification
* topic modeling
* sentiment analysis
